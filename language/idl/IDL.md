# 接口描述语言（Interface Description language）
接口描述语言或者接口定义语言（Interface Definition language）是一种特殊的语言，用于以语言无关的方式描述软件部件间的应用程序接口（API)。可以让运行与不同操作系统，使用不同程序设计语言编写的软件进行通讯。

通常每种IDL语言都会提供工具将使用IDL定义的接口转换成相应的程序设计语言。
