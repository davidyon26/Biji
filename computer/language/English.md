## 标点符号
| 符号  | 英文 | 中文 |
|----- |----- |-----|
| \|  | pipes | 竖线 |
| -   | hyphens | 连字符 |
| `   | backtick| 反引号 |
| '   | single quotation mark<br> | 单引号 |
| *   | asterisk| 星号 |
| _   | underline <br> underscore | 下划线 |
| ^   | carets | 插入符号 |
| ~   | tildes | 波浪号  |
| :  | colons | 冒号 |
| ;  | semi-colons | 分号 |
